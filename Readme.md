# Folder Structure

```yaml
terraform/
│── environments/
│   ├── dev/
│   │   ├── main.tf          # Your main Terraform code for dev environment
│   │   ├── providers.tf     # AWS provider configuration
│   │   ├── terraform.tfvars # Variable values
│   │   ├── variables.tf     # Declare the bucket_name variable

```
